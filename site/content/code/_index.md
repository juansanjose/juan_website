Upload your code snippets, scripts, and utilities to share with others. All uploads are stored locally and can be browsed below.

## How to Upload

1. Fill out the upload form on the home page
2. Provide a title and description
3. Select your code file
4. Click Upload

Your code will be available for preview and download.
